# FDF

This program can parse a file with xyz-coordinates of a point and create a simplified 3D graphic representation of a relief landscape.

Usage:

(simple mode)\
./fdf \<filename>

(mode with filled polygon and z-buffer)\
./fdf \<filename> -f
